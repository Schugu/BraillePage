El sistema Braille

Es un sistema de lectura y escritura que se vale de puntos en relieve para representar todo lo que las personas ciegas necesitan escribir.

El signo básico del Sistema se denomina SIGNO GENERADOR, y está formado por seis puntos.

Combinando de distintas maneras estos seis puntos, pueden representarse todas las letras, los números, los signos, la musicografía, etc.

Para facilitar la identificación de cada punto, cada uno, toma el nombre de un número.

El alfabeto

Las primeras diez combinaciones, que representan las diez primeras letras del alfabeto, ocupan sólo los cuatro puntos superiores del signo generador. Este grupo de signos, se denomina PRIMERA SERIE.

Cada una de las siguientes diez combinaciones, agrega a las de la primera serie, el punto número 3. Este grupo de signos, se denomina SEGUNDA SERIE.

Las siguientes cinco combinaciones, agregan el punto número 6, a las de la segunda serie. Este grupo de signos, se denomina TERCERA SERIE.

Estas dos letras, completan el alfabeto.

Los números

Un signo formado por los puntos 3, 4, 5 y 6. A ese signo se lo denomina SIGNO DE NUMERO.

Los dígitos se representan anteponiendo a las diez primeras letras del alfabeto.

Cuando se necesite escribir cantidades de mas de una cifra, se antepondrá sólo un signo de número a toda la cantidad, sin importar cuántas cifras que lleve. El efecto del signo de número para convertir los signos de las letras, en números, termina al dejar un espacio en blanco.
